# バックアップ/DR運用

## 目的
バックアップとDR手順を運用に落とし込み、RTO/RPOを達成する。

## 対応NFR
- NFR-AV-005, AV-006, AV-009
- NFR-PRC-005
- NFR-COST-005

## バックアップ運用
- 詳細設定は DD-BackupRestore-DR.md に準拠
- 日次/週次バックアップの成功率を監視
- 失敗時は24時間以内に再実行

## 復元訓練
- 四半期に1回、復元手順を実施
- 検証観点: 件数/参照整合性/主要集計
- 訓練記録を保存（PRC-008）

## DR訓練
- 年2回実施
- 切替/戻し手順を記録
- RTO/RPOの実測値を保存

## データ所在地
- 制約地域は同地域内DRを優先
- 例外時は法務承認と暗号化を必須
