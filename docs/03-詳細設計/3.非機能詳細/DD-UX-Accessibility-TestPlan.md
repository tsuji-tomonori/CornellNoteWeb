# アクセシビリティ検証計画（詳細）

## 対応NFR
NFR-UX-003〜026、NFR-UX-023/024/025中心

## 自動検査（CIゲート）
- ツール: axe-core + Lighthouse
- 対象ページ: ログイン/ノート一覧/ノート詳細/共有/設定
- 重大度閾値
  - 重大/深刻: 0件
  - 中: 10件以内（リリース前に是正計画必須）
- レポート保存: 毎リリース

## 手動検査
- キーボードのみで主要フロー完遂
- スクリーンリーダー組合せ
  - NVDA + Firefox
  - VoiceOver + Safari
- 200%拡大/320px幅/コントラスト
  - コントラスト比 4.5:1以上

## 外部サービス評価（UX-024）
- VPAT取得（未取得の場合は代替評価）
- 主要フロー検査と緩和策記録

## 不具合運用（UX-025）
- 受付窓口: support@company
- 重大度定義
  - Critical: 操作不能
  - High: 主要フロー阻害
  - Medium: 回避可能
- 是正期限
  - Critical: 7日
  - High: 14日
  - Medium: 30日
- 回避策提示: 48時間以内
