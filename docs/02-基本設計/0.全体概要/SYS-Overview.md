# システム概要

## 目的
Cornellノート形式で学習ノートを作成・保存・検索できるWebアプリを提供する。

## 対象範囲
- ノート作成/編集/保存/一覧/検索
- アカウント登録・ログイン・退会
- 共有リンク（閲覧専用）
- PDFエクスポート

## 外部連携
- 外部IdP（Google OAuth）
- メール送信サービス（パスワードリセット）
- PDF生成方式（サーバー生成 or クライアント生成のどちらかを採用）

## 前提・非対象
- Webアプリとして提供（PC-001）
- 対応ブラウザは最新2メジャー（PC-002）
- ユーザーごとのデータ分離（PC-003）
- 共有リンクは閲覧専用（PC-004）
- Google OAuthは任意（PC-005）
- メール送信は外部委譲（PC-006）
- MVPはコア機能に限定（PC-007）
- 共同編集、リアルタイム同期、オフライン編集は対象外
