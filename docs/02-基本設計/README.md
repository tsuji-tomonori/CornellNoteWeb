# 基本設計（HLD）

本フォルダは、要件に対して「どういう構造で実現するか」を合意するための高位設計をまとめる。
実装着手前の設計レビュー対象とする。

## 構成
- `docs/02-基本設計/0.全体概要`: システム概要、コンテキスト、前提/非対象
- `docs/02-基本設計/1.アーキテクチャ`: 論理構成、デプロイ構成、主要データフロー
- `docs/02-基本設計/2.データ設計`: ドメインモデル、ERD、保持・削除方針
- `docs/02-基本設計/3.UI設計`: 画面一覧、遷移、Cornellレイアウト
- `docs/02-基本設計/4.API設計`: エンドポイント概要、認証、エラー体系
- `docs/02-基本設計/5.非機能設計`: セキュリティ、性能、監視/運用
