# PC-006 メール送信は外部サービスを利用する

## Snow Card
- **Requirement ID**: PC-006
- **Type**: Project Constraint
- **Description（INCOSE形式）**: 本システムは、パスワードリセット等のメール送信を外部メール送信サービスに委譲しなければならない。
- **Rationale**: 自前SMTP運用のコストと到達率リスクを避けるため。
- **Fit Criterion（確認方法）**: パスワードリセット要求から、外部サービス経由でメールが送信されることを検証環境で確認する。
- **Customer Satisfaction**: 2 / 5
- **Customer Dissatisfaction**: 3 / 5
- **Priority**: Should
- **Originator**: 運用担当
- **Supporting Materials**: メール送信サービス選定資料、SPF/DKIM設定
- **Conflicts/Dependencies**: 外部サービスの制限（送信上限、停止時の復旧）に依存。
