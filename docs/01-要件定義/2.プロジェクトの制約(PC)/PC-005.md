# PC-005 外部認証はGoogle OAuthを任意で利用可能とする

## Snow Card
- **Requirement ID**: PC-005
- **Type**: Project Constraint
- **Description（INCOSE形式）**: 本システムは、メール/パスワード認証に加えて、Google OAuth によるログインを任意で提供できるようにしなければならない。
- **Rationale**: パスワード管理負荷を下げ、初回登録の摩擦を減らすため。
- **Fit Criterion（確認方法）**: Google アカウントでログインでき、同一ユーザーとしてノート領域にアクセスできることを確認する（UC-01 A02）。
- **Customer Satisfaction**: 3 / 5
- **Customer Dissatisfaction**: 2 / 5
- **Priority**: Could
- **Originator**: プロダクトオーナー
- **Supporting Materials**: OAuth設定手順、Google側アプリ登録
- **Conflicts/Dependencies**: 外部IdP障害時の取り扱い（代替ログイン手段）をPIで管理する。
