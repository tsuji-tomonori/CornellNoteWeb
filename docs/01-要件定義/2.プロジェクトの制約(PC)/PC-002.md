# PC-002 対応ブラウザを限定する

## Snow Card
- **Requirement ID**: PC-002
- **Type**: Project Constraint
- **Description（INCOSE形式）**: 本システムは、サポート対象ブラウザを Chrome/Edge/Safari/Firefox の最新2メジャーバージョンに限定しなければならない。
- **Rationale**: 品質担保とテストコストを現実的にするため。
- **Fit Criterion（確認方法）**: リリース時点で対象ブラウザ一覧を公開し、E2Eテストを対象ブラウザで実施した結果を証跡として残す。
- **Customer Satisfaction**: 3 / 5
- **Customer Dissatisfaction**: 3 / 5
- **Priority**: Should
- **Originator**: 開発リード
- **Supporting Materials**: 対応ブラウザ一覧、E2Eテストレポート
- **Conflicts/Dependencies**: 古いブラウザ利用者の要望とはトレードオフ。
