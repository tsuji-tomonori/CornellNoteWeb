# PP-001 コーネル式ノート作成体験を提供する

## Snow Card
- **Requirement ID**: PP-001
- **Type**: Project Purpose
- **Description（INCOSE形式）**: 本プロジェクトは、利用者がブラウザ上でコーネル式ノート（Cue/Notes/Summary）を作成・保存できる環境を提供しなければならない。
- **Rationale**: コーネル式は記録と復習を分離でき、学習定着に寄与するため。形式を固定した作成体験を提供することが主要価値となる。
- **Fit Criterion（成功条件/検証）**: ベータ利用者（10名以上）が、初回ログインから5分以内に UC-02（新規ノート作成）を完了でき、作成したノートが再ログイン後も保持されることをデモ/受入試験で確認する。
- **Customer Satisfaction**: 5 / 5
- **Customer Dissatisfaction**: 5 / 5
- **Priority**: Must
- **Originator**: プロダクトオーナー
- **Supporting Materials**: UC-02/UC-03、画面モック、オンボーディング案
- **Conflicts/Dependencies**: 認証（FR/SEC）・データ永続化（FR/DATA）に依存。
