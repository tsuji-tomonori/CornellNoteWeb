# FR-016 編集内容を保存できる

## Snow Card
- **Requirement ID**: FR-016
- **Type**: Functional
- **Use Case**: UC-03
- **Description（INCOSE形式）**: 本システムは、利用者の編集内容（Cue/Notes/Summary/タイトル等）を永続化できるようにしなければならない。
- **Rationale**: 保存できない場合に製品価値が成立しないため。
- **Fit Criterion（受入基準/検証）**: 編集後にブラウザを再読み込みしても内容が失われないことを確認する（テスト）。
- **Customer Satisfaction**: 5 / 5
- **Customer Dissatisfaction**: 5 / 5
- **Priority**: Must
- **Originator**: プロダクトオーナー
- **Supporting Materials**: UC-03、データモデル案
- **Conflicts/Dependencies**: データ保持方針（NFR-DATA）と整合が必要。
