| FR-011 | Cornellレイアウト（Cue/Notes/Summary）を表示できる | UC-02, UC-03 | Must |
| FR-012 | Cue領域へテキスト入力できる | UC-03 | Must |
| FR-013 | Notes領域へテキスト入力できる | UC-03 | Must |
| FR-014 | Summary領域へテキスト入力できる | UC-03 | Must |
| FR-015 | ノートタイトルを編集できる | UC-02, UC-03 | Must |
| FR-016 | 編集内容を保存できる | UC-03 | Must |
| FR-017 | 保存状態（保存済み/未保存/失敗）を表示できる | UC-03 | Should |
| FR-020 | ノート一覧を表示できる | UC-02, UC-03, UC-05 | Must |
| FR-021 | ノートブックを作成できる | UC-04 | Should |

| FR-040 | CornellレイアウトでPDFを生成できる | UC-06 | Should |
