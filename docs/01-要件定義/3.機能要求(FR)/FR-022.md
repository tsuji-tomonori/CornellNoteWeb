# FR-022 ノートにタグを付与できる

## Snow Card
- **Requirement ID**: FR-022
- **Type**: Functional
- **Use Case**: UC-04
- **Description（INCOSE形式）**: 本システムは、利用者がノートにタグを付与・削除できるようにしなければならない。
- **Rationale**: 横断的な整理軸が必要なため。
- **Fit Criterion（受入基準/検証）**: タグ付与後、ノート表示にタグが表示され、タグ絞り込み（UC-05 A01）で該当ノートが表示されることを確認する（テスト）。
- **Customer Satisfaction**: 4 / 5
- **Customer Dissatisfaction**: 3 / 5
- **Priority**: Should
- **Originator**: エンドユーザー代表
- **Supporting Materials**: UC-04/05、タグ設計案
- **Conflicts/Dependencies**: 検索/絞り込み（FR-030）と整合が必要。
