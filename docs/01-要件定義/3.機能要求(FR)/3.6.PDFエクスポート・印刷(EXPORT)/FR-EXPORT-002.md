# FR-EXPORT-002 印刷用表示を提供できる

## Snow Card
- **Requirement ID**: FR-EXPORT-002
- **Type**: Functional
- **Use Case**: UC-06

- **Customer Satisfaction**: 3 / 5
- **Customer Dissatisfaction**: 2 / 5
- **Priority**: Could
- **Originator**: エンドユーザー代表
- **Supporting Materials**: UC-06
