# 3.1 ユースケース概要（一覧・境界・主要アクター）

## 3.1.1 製品境界（Product Boundary）

**製品名（仮）**：CornellNote Web

### In Scope（本システムが担うこと）
- ブラウザ上で Cornell 形式（Cue/Notes/Summary）のノートを作成・編集・保存できること
- ノートを整理（ノートブック、タグ）し、検索できること
- ノートを PDF にエクスポート／印刷できること
- 共有リンク（閲覧のみ）で第三者にノートを共有できること
- ユーザー認証（メール/パスワード、任意で OAuth）とアカウント管理を提供すること

### Out of Scope（本システムが担わないこと）
- 学習教材（講義動画、問題集等）の提供
- 手書き認識・OCR
- LMS 連携（採点、出席等）
- ネイティブアプリ専用機能（※PWAは将来検討）

### 外部システム境界（External Systems）
- ブラウザ（Chrome/Edge/Safari/Firefox）
- 外部認証プロバイダ（Google 等の OAuth、任意）
- メール送信サービス（パスワードリセット等）

---

## 3.1.2 主要アクター（Actors）

- **A1: ノート作成者（Registered User）**  
  自分のノートを作成・編集・整理・検索・エクスポートする。
- **A2: 共有閲覧者（Shared Viewer）**  
  共有リンク経由でノートを閲覧する（編集不可）。
- **A3: システム管理者（Admin）**  
  アカウント停止等の運用管理を行う（MVPでは任意）。
- **A4: 外部認証プロバイダ（IdP）**  
  OAuth 認証結果を返す（外部システム）。
