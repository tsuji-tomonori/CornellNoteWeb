# NFR-INT-002 データエクスポート（ポータビリティ）

## Snow Card
- **Requirement ID**: NFR-INT-002
- **Type**: Non-functional（Portability）
- **Description（INCOSE形式）**: 本システムは、主要業務データを、合意された標準フォーマット（例：CSV/JSON）でエクスポートできなければならない。
- **Rationale**: データポータビリティとベンダーロックイン低減のため。
- **Fit Criterion（受入基準/検証）**: エクスポート対象・形式・頻度が定義され、サンプル出力の妥当性が確認できること（テスト＋検査）。
- **Customer Satisfaction**: 3 / 5
- **Customer Dissatisfaction**: 4 / 5
- **Priority**: Could
- **Originator**: 事業部門／監査
- **Supporting Materials**: データ台帳、エクスポート仕様
- **Conflicts/Dependencies**: 個人情報削除（NFR-PRC-003）と整合が必要。
