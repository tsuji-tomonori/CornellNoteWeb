# NFR-UX-002 ブラウザ/デバイス互換性

## Snow Card
- **Requirement ID**: NFR-UX-002
- **Type**: Non-functional（Compatibility/Usability）
- **Description（INCOSE形式）**: 本システムは、合意したサポート対象ブラウザおよびデバイスで、主要機能が同等に利用できなければならない。
- **Rationale**: 利用者環境の差による業務停止を防ぐため。
- **Fit Criterion（受入基準/検証）**: サポート対象（例：最新2世代の主要ブラウザ）を定義し、主要ユースケースの動作確認結果をリリースごとに保管する（テスト＋検査）。
- **Customer Satisfaction**: 3 / 5
- **Customer Dissatisfaction**: 3 / 5
- **Priority**: Should
- **Originator**: プロダクト／QA
- **Supporting Materials**: サポートマトリクス、テスト結果
- **Conflicts/Dependencies**: 端末制約が強い場合は対象範囲を明確化する。
