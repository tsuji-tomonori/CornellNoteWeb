# UL-014 p95

## Snow Card
- **Term ID**: UL-014
- **Type**: Ubiquitous Language
- **Term（正）**: p95（95パーセンタイル）
- **Definition**: 計測対象値を小さい順に並べたとき、全体の95%がその値以下となる境界値。性能要件の指標として用いる。
- **Includes**: 性能指標 / 測定区間/期間の明示
- **Excludes**: 平均値（mean）のみでの評価
- **Synonyms/Aliases**: P95 / 95th percentile
- **Example**: API応答時間の p95 を 300ms 以下とする
- **Originator**: 要件定義
- **Related Terms**: なし
