# UL-010 オーナー

## Snow Card
- **Term ID**: UL-010
- **Type**: Ubiquitous Language
- **Term（正）**: オーナー（Owner）
- **Definition**: ノートの所有者。ノートの作成・編集・削除・共有設定の変更を行える主体。
- **Includes**: 登録ユーザー / ノートの編集権限
- **Excludes**: 共有閲覧者（Viewer）
- **Synonyms/Aliases**: 作成者
- **Example**: ノートを作成したユーザー
- **Originator**: 要件定義
- **Related Terms**: UL-005 / UL-011
