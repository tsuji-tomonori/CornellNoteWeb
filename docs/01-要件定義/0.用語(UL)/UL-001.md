# UL-001 Cornellノート

## Snow Card
- **Term ID**: UL-001
- **Type**: Ubiquitous Language
- **Term（正）**: Cornellノート
- **Definition**: コーネル式ノート術の形式で記述された、Cue/Notes/Summary の3領域を持つ1つのノート。
- **Includes**: Cue領域 / Notes領域 / Summary領域 / タイトル / 作成者（オーナー） / 作成日時/更新日時
- **Excludes**: 複数ノートの束（ノートブック） / 外部資料（講義動画、教材本文）そのもの
- **Synonyms/Aliases**: コーネルノート / コーネル式ノート
- **Example**: 講義1回分のノートを Cornell 形式で1枚にまとめたもの
- **Originator**: 要件定義
- **Related Terms**: UL-002 / UL-003 / UL-004 / UL-005
