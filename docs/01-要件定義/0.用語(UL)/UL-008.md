# UL-008 共有リンク

## Snow Card
- **Term ID**: UL-008
- **Type**: Ubiquitous Language
- **Term（正）**: 共有リンク（Share Link）
- **Definition**: 特定ノートを閲覧専用で共有するために発行されるURL。共有トークンを含み得る。
- **Includes**: 閲覧専用アクセス / 共有トークン / 共有の有効/無効状態
- **Excludes**: 編集権限の委譲 / 共同編集（将来機能）
- **Synonyms/Aliases**: 共有URL / 公開リンク
- **Example**: URL を送付して第三者がブラウザで閲覧する
- **Originator**: 要件定義
- **Related Terms**: UL-009 / UL-011 / UL-005
