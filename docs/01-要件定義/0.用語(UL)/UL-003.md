# UL-003 Notes領域

## Snow Card
- **Term ID**: UL-003
- **Type**: Ubiquitous Language
- **Term（正）**: Notes（ノーツ）
- **Definition**: Cornellノートの本文領域。講義・会議などで得た情報を時系列/構造化して記録する。
- **Includes**: 本文テキスト / 箇条書き / 引用（短い）
- **Excludes**: Cue（キーワード） / Summary（要約）
- **Synonyms/Aliases**: 本文 / メインノート
- **Example**: 講義内容を箇条書きで記録する
- **Originator**: 要件定義
- **Related Terms**: UL-001 / UL-002 / UL-004
