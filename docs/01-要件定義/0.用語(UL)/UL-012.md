# UL-012 自動保存

## Snow Card
- **Term ID**: UL-012
- **Type**: Ubiquitous Language
- **Term（正）**: 自動保存（Autosave）
- **Definition**: ユーザーの編集操作に応じて、明示的な保存操作なしに編集内容を自動的に永続化する仕組み。
- **Includes**: 一定間隔の保存 / 編集イベント単位の保存 / 保存失敗の検知
- **Excludes**: 保存が保証されない一時メモ（下書きのみで永続化しない状態）
- **Synonyms/Aliases**: Autosave
- **Example**: 編集後数秒以内に保存状態が「保存済み」になる
- **Originator**: 要件定義
- **Related Terms**: UL-013 / UL-005
