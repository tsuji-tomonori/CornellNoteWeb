# UL-011 共有閲覧者

## Snow Card
- **Term ID**: UL-011
- **Type**: Ubiquitous Language
- **Term（正）**: 共有閲覧者（Shared Viewer）
- **Definition**: 共有リンクを通じてノートを閲覧する主体。原則として編集はできない。
- **Includes**: 閲覧操作 / 共有リンクへのアクセス
- **Excludes**: オーナー権限 / 編集操作
- **Synonyms/Aliases**: Viewer / 閲覧者
- **Example**: 先生が学生の共有リンクを開いて内容を確認する
- **Originator**: 要件定義
- **Related Terms**: UL-008 / UL-010
