# UL-005 ノート

## Snow Card
- **Term ID**: UL-005
- **Type**: Ubiquitous Language
- **Term（正）**: ノート（Note）
- **Definition**: 本システムに保存される1つのノートデータ。基本は Cornellノート形式だが、識別子やメタデータ（所有者、分類、共有状態等）を含む。
- **Includes**: Cornell ノート本文 / タイトル / ノートブック / タグ / 共有設定 / 作成/更新日時
- **Excludes**: ノートブック（ノート集合）
- **Synonyms/Aliases**: Note
- **Example**: UC-03 で編集・保存される対象
- **Originator**: 要件定義
