# UL-006 ノートブック

## Snow Card
- **Term ID**: UL-006
- **Type**: Ubiquitous Language
- **Term（正）**: ノートブック（Notebook）
- **Definition**: 複数のノートをまとめるための分類単位。科目/プロジェクト/テーマ等でノートを整理する。
- **Includes**: ノートの集合 / ノートブック名
- **Excludes**: タグ（横断ラベル）
- **Synonyms/Aliases**: Notebook / フォルダ
- **Example**: 『線形代数』ノートブックに関連ノートを格納する
- **Originator**: 要件定義
- **Related Terms**: UL-005 / UL-007
