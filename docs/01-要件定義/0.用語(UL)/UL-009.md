# UL-009 共有トークン

## Snow Card
- **Term ID**: UL-009
- **Type**: Ubiquitous Language
- **Term（正）**: 共有トークン（Share Token）
- **Definition**: 共有リンクに含まれるランダムな識別子（ベアラートークン）。リンクを知っている者が閲覧できる方式で用いる。
- **Includes**: 推測困難な文字列 / 共有リンクの識別
- **Excludes**: ユーザーの認証情報（パスワード等）
- **Synonyms/Aliases**: 共有キー / トークン
- **Example**: 共有URLの末尾に付与されるランダム文字列
- **Originator**: 要件定義
- **Related Terms**: UL-008
