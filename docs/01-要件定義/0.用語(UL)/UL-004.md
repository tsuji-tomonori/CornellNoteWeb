# UL-004 Summary領域

## Snow Card
- **Term ID**: UL-004
- **Type**: Ubiquitous Language
- **Term（正）**: Summary（サマリー）
- **Definition**: Cornellノートの下部に配置される要約領域。Notes 全体を自分の言葉で短く要約する。
- **Includes**: 要約文 / 次回のアクション/学びのまとめ
- **Excludes**: Notes の逐語記録 / 別ノートへのリンク集（将来機能）
- **Synonyms/Aliases**: まとめ / 要約欄
- **Example**: Notes を 3〜5行で要約する
- **Originator**: 要件定義
- **Related Terms**: UL-001 / UL-002 / UL-003
