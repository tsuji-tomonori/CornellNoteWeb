# UL-002 Cue領域

## Snow Card
- **Term ID**: UL-002
- **Type**: Ubiquitous Language
- **Term（正）**: Cue（キュー）
- **Definition**: Cornellノートの左側（または左カラム）に配置される領域。キーワード、問い、重要語、想起の手がかりを記録する。
- **Includes**: 質問 / キーワード / 重要語 / 用語の定義メモ
- **Excludes**: 本文の詳細記述（Notes） / 要約（Summary）
- **Synonyms/Aliases**: キュー / Cue Column / キーワード欄
- **Example**: Notes の段落ごとに対応する質問を Cue に書く
- **Originator**: 要件定義
- **Related Terms**: UL-001 / UL-003 / UL-004
