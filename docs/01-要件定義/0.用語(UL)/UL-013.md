# UL-013 保存状態

## Snow Card
- **Term ID**: UL-013
- **Type**: Ubiquitous Language
- **Term（正）**: 保存状態（Save Status）
- **Definition**: ノート編集画面で表示される、保存に関する状態表示。例：保存済み／未保存／保存失敗。
- **Includes**: 保存済み / 未保存 / 保存失敗 / 再試行可能
- **Excludes**: 品質指標としての性能値（p95等）
- **Synonyms/Aliases**: ステータス表示
- **Example**: 通信断で保存失敗した際に「保存失敗」と表示される
- **Originator**: 要件定義
- **Related Terms**: UL-012
