# UL-007 タグ

## Snow Card
- **Term ID**: UL-007
- **Type**: Ubiquitous Language
- **Term（正）**: タグ（Tag）
- **Definition**: ノートに付与できる横断的なラベル。検索や絞り込みのキーとして利用する。
- **Includes**: タグ名 / ノートへの多対多関連
- **Excludes**: ノートブック（階層的な分類）
- **Synonyms/Aliases**: Tag / ラベル
- **Example**: 『試験範囲』タグを複数ノートに付与する
- **Originator**: 要件定義
- **Related Terms**: UL-005 / UL-006
